Linear Algebra Midterm Examination

Question 1 (20 points)

Let A be the matrix:

A = [ 2 -1 3 ]
[ 1 0 -2 ]
[ 3 1 1 ]

a) Find the determinant of A.

b) Find A⁻¹ (the inverse of A).

c) Solve the system Ax = b where b = [1, 0, 2]ᵀ.

Question 2 (25 points)

Let V = {[x₁, x₂, x₃]ᵀ ∈ ℝ³ : x₁ + 2x₂ - x₃ = 0}.

a) Show that V is a subspace of ℝ³.

b) Find a basis for V.

c) What is the dimension of V?

d) Find the orthogonal complement V⊥.

Question 3 (25 points)

Consider the linear transformation T: ℝ³ → ℝ² defined by:

T([x₁, x₂, x₃]ᵀ) = [2x₁ - x₂ + x₃, x₁ + 3x₂]ᵀ

a) Find the standard matrix representation of T.

b) Find the kernel of T (null space).

c) Find the range of T (column space).

d) Determine if T is one-to-one and/or onto.

Question 4 (20 points)

Find the eigenvalues and eigenvectors of the matrix:

B = [ 3 1 ]
[ 2 4 ]

Is B diagonalizable? If so, find a diagonal matrix D and an invertible matrix P such that B = PDP⁻¹.

Question 5 (10 points)

Let u = [1, 2, -1]ᵀ and v = [3, 0, 1]ᵀ.

a) Compute u · v (dot product).

b) Compute ||u|| and ||v|| (norms).

c) Find the angle between u and v.

d) Compute u × v (cross product).

Useful Formulas:

• det([a b; c d]) = ad - bc
• For 3×3 matrix, det(A) = a₁₁(a₂₂a₃₃ - a₂₃a₃₂) - a₁₂(a₂₁a₃₃ - a₂₃a₃₁) + a₁₃(a₂₁a₃₂ - a₂₂a₃₁)
• u · v = ||u|| ||v|| cos(θ)
• Eigenvalues satisfy det(A - λI) = 0
