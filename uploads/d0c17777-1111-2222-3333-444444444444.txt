Assignment Summary:
Explore techniques for transforming raw datasets into meaningful input features.

Key Topics:
- Feature selection (manual, variance threshold, correlation)
- Feature extraction (PCA, t-SNE)
- Encoding (one-hot, label encoding)
- Normalization (min-max, z-score)

Example Tasks:
- Remove redundant columns
- Reduce dimensions using PCA
- One-hot encode categorical variables

Tips:
- Visualize correlations before selecting features
- Check for multicollinearity
- Normalize before applying distance-based algorithms

Common Pitfalls:
- Applying PCA before normalization
- Encoding leakage (target leakage)
- Not splitting train/test before transformation
