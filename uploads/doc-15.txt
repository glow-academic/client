// Network Security Assignment: Secure Protocols

Assignment Summary:
Study and simulate encryption protocols such as TLS and symmetric/asymmetric encryption.

Topics:
- Symmetric (AES) vs Asymmetric (RSA, Diffie-Hellman)
- Handshake protocols and session keys
- Digital signatures and certificates

Example Task:
Simulate a secure message exchange using:
1. RSA key pair for identity
2. AES session key for encrypted chat

Concept Reinforcements:
- Public vs private key distinction
- Importance of nonce/randomness
- Threats: man-in-the-middle, replay attacks
