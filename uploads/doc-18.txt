Assignment Summary:
Understand and compute classification performance metrics.

Metrics:
- Accuracy: (TP + TN) / Total
- Precision: TP / (TP + FP)
- Recall: TP / (TP + FN)
- F1-Score: 2 * (Precision * Recall) / (Precision + Recall)
- Confusion Matrix analysis

Tasks:
- Compute confusion matrix from predictions
- Analyze imbalanced dataset performance
- Choose appropriate metric (e.g., Recall for medical diagnosis)

Tips:
- Precision vs Recall trade-off: tune thresholds
- Use ROC/AUC for binary classifiers
- Don’t rely solely on accuracy for imbalanced classes
