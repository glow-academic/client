Assignment Summary:
Prove the time complexity of recursive algorithms using the Master Theorem or recurrence tree method.

Examples:
1. T(n) = 2T(n/2) + n → O(n log n) (Merge Sort)
2. T(n) = 3T(n/2) + n² → Use Master Theorem, case 1–3 analysis
3. T(n) = T(n − 1) + 1 → O(n)

Steps:
- Identify a, b, f(n) in T(n) = aT(n/b) + f(n)
- Compare f(n) to n^log_b(a)
- Apply correct Master Theorem case or expand with tree

Pitfalls:
- Misapplying base case conditions
- Forgetting to check regularity condition
- Not simplifying logarithmic expressions properly
