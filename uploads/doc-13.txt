Assignment Summary:
Implement a simplified TCP/IP stack that includes basic functionality from the transport and network layers.

Required Components:
- IP addressing and packet encapsulation
- TCP-style reliable transport (sequence numbers, ACKs)
- Simulated packet loss and retransmission logic

Concept Map:
1. Application Layer → sends data to Transport Layer
2. Transport Layer (TCP) → splits data into segments
3. Network Layer (IP) → routes packets

Tips:
- Use enums or constants for headers
- Simulate transmission with functions for send/receive
- Log each step of the packet's journey
