Regression Analysis of Housing Prices

Executive Summary

This project analyzes the relationship between housing prices and various predictor variables including square footage, number of bedrooms, location, and age of property. Using multiple linear regression techniques, we developed a model to predict housing prices and identified the most significant factors influencing property values.

Key Findings:

  • Square footage is the strongest predictor of housing price (coefficient: 125.50, p < 0.001)
  • Location (coded as dummy variables) shows significant variation in prices across different neighborhoods
  • The model explains 78% of the variance in housing prices (R² = 0.78)
  • Number of bedrooms shows a non-linear relationship with price

Introduction and Objectives

The objective of this analysis was to develop a predictive model for housing prices based on available property characteristics. Understanding the factors that influence housing prices has important implications for buyers, sellers, and real estate professionals.

Research Questions:

  1. Which property characteristics are most predictive of housing prices?
  2. How do location factors affect property values?
  3. Can we develop an accurate predictive model for housing prices?
  4. Are there any interactions between predictor variables?

Data Description

The dataset consists of 450 residential property sales from the past two years. Variables included:

Response Variable:

  • Price: Sale price in thousands of dollars (range: $150K - $850K)

Predictor Variables:

  • Square_Footage: Total living area in square feet (range: 800 - 4,200)
  • Bedrooms: Number of bedrooms (range: 1 - 5)
  • Bathrooms: Number of bathrooms (range: 1 - 4)
  • Age: Age of property in years (range: 0 - 45)
  • Location: Categorical variable with 5 neighborhoods (North, South, East, West, Central)
  • Garage: Binary variable (0 = no garage, 1 = has garage)

Methodology

Model Selection:

We employed multiple linear regression with the following approach:

  1. Exploratory data analysis including correlation matrices and scatter plots
  2. Initial model with all predictors
  3. Stepwise selection using AIC to identify the best model
  4. Diagnostic checking for assumptions (normality, homoscedasticity, independence)
  5. Model validation using cross-validation

Model Equation:

Price = β₀ + β₁(Square_Footage) + β₂(Bedrooms) + β₃(Bathrooms) + β₄(Age) + β₅(Garage) + β₆₋₁₀(Location_Dummies) + ε

Results

Final Model Summary:

After stepwise selection, the final model included square footage, bedrooms, bathrooms, age, garage, and location as significant predictors.

Coefficient Estimates:

  • Intercept: 45.32 (SE = 8.15, p < 0.001)
  • Square Footage: 0.125 (SE = 0.008, p < 0.001)
  • Bedrooms: 12.45 (SE = 3.22, p < 0.001)
  • Bathrooms: 18.67 (SE = 4.11, p < 0.001)
  • Age: -1.23 (SE = 0.15, p < 0.001)
  • Garage: 22.50 (SE = 5.33, p < 0.001)

Model Performance:

  • R² = 0.78 (Adjusted R² = 0.76)
  • RMSE = 42.3 thousand dollars
  • F-statistic = 156.8 (p < 0.001)

Diagnostic Checks:

Residual analysis showed:

  • Residuals are approximately normally distributed (Shapiro-Wilk test: p = 0.18)
  • No clear patterns in residual plots, suggesting homoscedasticity
  • No significant multicollinearity (VIF values all < 3.0)

Discussion

The analysis revealed that square footage is the most important predictor of housing price, with each additional square foot associated with a $125 increase in price. This makes intuitive sense as larger homes typically command higher prices.

Location effects were significant, with properties in the North and West neighborhoods showing premiums of $35K and $28K respectively compared to the Central baseline. The South and East neighborhoods showed discounts, suggesting these areas may have lower desirability or different market conditions.

The negative coefficient for age (-1.23) indicates that older properties sell for less, though the effect is relatively modest at about $1,230 per year of age.

Limitations:

  • The model is based on historical data and may not account for future market changes
  • Some potentially important factors (school quality, crime rates) were not available
  • The sample was limited to one metropolitan area

Conclusion

This analysis successfully identified key factors influencing housing prices and developed a predictive model with good explanatory power (R² = 0.78). The model can be used by real estate professionals to estimate property values, though it should be supplemented with local market knowledge and qualitative factors not captured in the quantitative model.

Future work could include:

  • Non-linear transformations for variables showing curvature
  • Interaction terms between predictors
  • Temporal analysis to account for market trends
  • Expansion to include additional geographic areas
