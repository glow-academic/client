Assignment Summary:
Implement a simple feedforward neural network from scratch (without ML libraries) and train it on sample data.

Required Components:
- Forward pass (matrix multiplication, activation functions)
- Backpropagation (gradient descent)
- Loss function (e.g., MSE for regression, Cross-Entropy for classification)

Tips:
- Organize code into layers (Input → Hidden → Output)
- Use numpy for matrix operations if allowed
- Track weight updates and check for exploding/vanishing gradients

Debugging Advice:
- Start with 1 hidden layer, small learning rate
- Print loss after each epoch to ensure it decreases
- Overfit on small data to check correctness before scaling
