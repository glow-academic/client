Problem 1

Let {aₙ} be a sequence of real numbers. Prove that if lim(n→∞) aₙ = L, then lim(n→∞) |aₙ| = |L|.

Proof:

Given that lim(n→∞) aₙ = L, we know that for every ε > 0, there exists N ∈ ℕ such that for all n ≥ N, we have |aₙ - L| < ε.

We want to show that lim(n→∞) |aₙ| = |L|, which means for every ε > 0, there exists M ∈ ℕ such that for all n ≥ M, we have ||aₙ| - |L|| < ε.

Using the reverse triangle inequality: ||aₙ| - |L|| ≤ |aₙ - L|

Since lim(n→∞) aₙ = L, for any ε > 0, there exists N such that |aₙ - L| < ε for all n ≥ N.

Therefore, for all n ≥ N:

||aₙ| - |L|| ≤ |aₙ - L| < ε

Hence, lim(n→∞) |aₙ| = |L|. □

Problem 3

Prove that if f is continuous on [a,b] and differentiable on (a,b), and f(a) = f(b), then there exists c ∈ (a,b) such that f'(c) = 0.

Proof:

This is a statement of Rolle's Theorem. Here is the proof:

Since f is continuous on the closed interval [a,b], by the Extreme Value Theorem, f attains both its maximum and minimum on [a,b].

Case 1: If the maximum or minimum occurs at an interior point c ∈ (a,b), then by Fermat's Theorem, f'(c) = 0 (since c is a local extremum in an open interval).

Case 2: If both the maximum and minimum occur at the endpoints a and b, then since f(a) = f(b), we have f(x) = f(a) = f(b) for all x ∈ [a,b]. This means f is constant, so f'(x) = 0 for all x ∈ (a,b), and we can choose any c ∈ (a,b).

In either case, there exists c ∈ (a,b) such that f'(c) = 0. □

Problem 4

Let f: ℝ → ℝ be uniformly continuous. Prove that if {xₙ} is a Cauchy sequence, then {f(xₙ)} is also a Cauchy sequence.

Proof:

Let {xₙ} be a Cauchy sequence. We want to show that {f(xₙ)} is a Cauchy sequence, i.e., for every ε > 0, there exists N ∈ ℕ such that for all m,n ≥ N, we have |f(xₘ) - f(xₙ)| < ε.

Since f is uniformly continuous, for every ε > 0, there exists δ > 0 such that for all x,y ∈ ℝ with |x - y| < δ, we have |f(x) - f(y)| < ε.

Since {xₙ} is Cauchy, for the δ found above, there exists N ∈ ℕ such that for all m,n ≥ N, we have |xₘ - xₙ| < δ.

Therefore, for all m,n ≥ N:

|xₘ - xₙ| < δ ⟹ |f(xₘ) - f(xₙ)| < ε

Hence, {f(xₙ)} is a Cauchy sequence. □
